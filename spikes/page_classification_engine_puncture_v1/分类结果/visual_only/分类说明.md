# visual_only：纯视觉或空白页

## 定义

页面没有报告身份、导航或实质正文功能，只包含照片、装饰过渡视觉或真正空白内容。

## 正例与反例

- 正例：整页建筑照片，没有标题、口号、正文、目录、表格或图表。
- 正例：真正空白页或纯装饰纹理页。
- 反例：图片内可见完整审计正文，页面角色仍应判 `body`。
- 反例：图片内有报告名或章节标题，可能是 `cover`。
- 反例：品牌落款或联系信息承担结束功能，可能是 `end`。

## 筛选机制

原生文字为零只会使工程规则返回不确定，不能直接判纯视觉。千问必须查看截图，排除栅格正文、封面身份和结束落款；不一致时进入一级复核。
