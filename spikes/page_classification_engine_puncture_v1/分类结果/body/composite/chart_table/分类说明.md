# body/composite/chart_table：图表与表格组合工具箱

## 定义

主体数据图表与主体表格共同承载页面信息。图表具有柱、线、扇区、环形、坐标轴或图例关系；表格具有明确行列、表头和单元格数据，两者都不能作为附件忽略。

## 正例与反例

- 正例：上半页为柱状图，下半页为与图表同等重要的明细表。
- 反例：表格旁只有装饰性小图标，应判 `body/table`。
- 反例：图表下只有几行来源说明，不构成表格。

## 筛选机制

工程证据提供表格区域和图形信号，千问确认真实图表与真实表格是否都占显著区域。工具调度锁定图表像素和内部标签，只处理独立 PDF 文字对象，并由表格工具处理单元格回填。
