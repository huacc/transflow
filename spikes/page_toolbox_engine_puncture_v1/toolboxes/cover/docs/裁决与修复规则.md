# cover：裁决与修复规则

## 分类专用 Finding

| Finding | 严重度 | 直接证据 | owner |
|---|---|---|---|
| `COVER_TEXT_OVERFLOW` | HARD | 任一允许字号无法装入 `allowed_bbox` | `cover_layout_planner` |
| `COVER_SAFE_REDACTION_REGION_NOT_FOUND` | HARD | 红删区域无法避开受保护字形 | `cover_pdf_renderer` |
| `COVER_LOCKED_OBJECT_CHANGED` | HARD | 锁定对象哈希变化且允许区外像素变化 | `cover_pdf_renderer` |
| `COVER_PROTECTED_TEXT_CHANGED` | HARD | 非空白字形缺失或逐字形坐标漂移 | `cover_quality_judge` |
| `COVER_SOURCE_TEXT_REMAINED` | HARD | 待替换源文字仍可提取 | `cover_quality_judge` |
| `COVER_TRANSLATED_TEXT_MISSING` | HARD | 应渲染译文未出现在目标框 | `cover_quality_judge` |
| `COVER_VISUAL_ANCHOR_DRIFT` | HARD | 左/右/中心锚点超出相对容差 | `cover_quality_judge` |
| `COVER_OUTSIDE_ALLOWED_REGION_CHANGED` | HARD | 声明文字区域外出现像素变化 | `cover_pdf_renderer` |
| `FONT_NOT_EMBEDDED` | HARD | 目标字体资源未嵌入 | `cover_pdf_renderer` |

## Product PASS

- 所有待译容器恰好处理一次；已有目标语言伴随行不重复绘制；
- 译文完整出现、无溢出、无重叠，稀疏标题锚点与层级保持；
- 受保护文字、背景、图片、Logo、颜色和装饰保持；
- 图片内文字不翻译、不覆盖；
- 视觉页按字节透传；
- 人工视觉复核不评价译文用词，只评价 P9 排版与视觉不变量。

## CAPABILITY FAILED

- 翻译服务配置缺失、响应无效或请求合同不匹配；
- 字体缺失、文字所有权不完整或无安全红删区域；
- 译文无法在允许区域和最小字号内排入。

## Repair 允许范围

当前没有运行时自由 repair loop。允许的确定性选择只有：相对安全区内的字号档位、行高、原左/中/右锚点，以及经结构证据确认的双语伴随行去重。不得根据译文里的公司名、已知标题或样本身份改变布局。

## Repair 接受

目标 Finding 改善、无新增硬问题、锁定对象保持、完整复裁通过时才接受；否则回滚。

遮挡判断不得仅依据 paint order 或图片 bbox；后绘内容覆盖只产生候选，只有临时移除对应字形后 2× 源页渲染逐像素一致，才能将文字归入 `occluded_object_ids`。
