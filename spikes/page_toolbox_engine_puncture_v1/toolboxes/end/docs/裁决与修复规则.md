# end：裁决与修复规则

## 分类专用 Finding

| Finding | 严重度 | 直接证据 | owner |
|---|---|---|---|
| `END_LOCKED_OBJECT_CHANGED` | HARD | source/candidate 锁定对象指纹不同 | `end_pdf_renderer` |
| `END_PROTECTED_TEXT_CHANGED` | HARD | 受保护文字内容或 bbox 不再匹配 | `end_quality_judge` |
| `END_PASSTHROUGH_BYTES_CHANGED` | HARD | 无翻译页 source/candidate SHA-256 不同 | `end_pdf_renderer` |
| `END_OUTSIDE_ALLOWED_REGION_CHANGED` | HARD | 允许区外像素变化超过阈值 | `end_pdf_renderer` |
| `END_SOURCE_TEXT_REMAINED` | HARD | 被翻译原文仍在原 bbox | `end_quality_judge` |
| `END_TRANSLATED_TEXT_MISSING` | HARD | region 内未提取到完整译文 | `end_quality_judge` |
| `END_REQUIRED_LITERAL_MISSING` | HARD | 联系数字、链接或认证标识缺失 | `end_quality_judge` |
| `END_PROTECTED_ANCHOR_OVERLAP` | HARD | 译文安全区与保护 bbox 相交 | `end_quality_judge` |
| `END_TEXT_OVERFLOW` | HARD | 所有固定字号/行高 profile 均无法装入 | `end_layout_planner` |
| `FONT_NOT_EMBEDDED` | HARD | 候选未嵌入所需字体资源 | `end_pdf_renderer` |

## Product PASS

- 过程链完成，候选 PDF 存在；
- 视觉锁定对象和受保护文字不变；
- 原文无残留，译文及必保字面量完整；
- 译文不侵入保护锚点，允许区外无大范围变化；
- 透传页与源页字节等价。

## CAPABILITY FAILED

- 翻译 Provider、API key 或目标字体不可用；
- Provider 返回合同不完整或删除必保字面量；
- 红删安全区不存在。

## Repair 允许范围

当前不做渲染后 repair。布局只能在 region 已声明的 `allowed_bbox` 中调整字号和行高；不得移动 Logo、二维码、品牌名、URL，不能扩展到全页万能稀疏 planner。
