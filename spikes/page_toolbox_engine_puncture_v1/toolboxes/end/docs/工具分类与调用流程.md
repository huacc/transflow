# end：工具分类与调用流程

## 工具清单

| 工具 | 类别 | 输入 | 输出 | 失败语义 |
|---|---|---|---|---|
| `build_end_template` | 叶子模板 | `PageFacts`、语言方向 | translate/protect regions、局部安全区 | 安全区缺失时 capability fail |
| `build_end_translation_request` | 翻译合同 | 可翻译 regions | 单页 `PageTranslationRequest` | 无 region 时返回 `None` |
| `plan_end_layout` | 叶子布局 | 模板、译文、字体 | 固定锚点的局部 placement | 无法装入时 quality fail |
| `render_end_candidate` | 机械渲染 | 源页、模板、placement | 候选 PDF、渲染证据 | 锁定对象或保护文字改变时 process fail |
| `render_end_passthrough` | 机械透传 | 无可翻译文字的源页 | 字节等价候选 | 哈希不等时 process fail |
| `run_p10_page` | 固定编排 | 单页和 Provider | 自包含运行包、双裁决 | 显式 process/product/capability 终态 |

## 固定流程

```text
source snapshot
→ PageFacts
→ EndTemplateBuilder
→ [无可翻译 region] byte passthrough
→ [有可翻译 region] PageTranslationRequest → Provider → EndLayoutPlanner → PDF renderer
→ locked/protected/literal/residue/font/outside-region judge
→ PAGE_PASSED | QUALITY_FAILED | CAPABILITY_FAILED | PROCESS_FAILED
```

P10 不建设自由 repair loop。当前唯一适配是布局阶段在固定安全区内尝试有限字号和行高 profile；全部失败即 `END_TEXT_OVERFLOW`，不移动视觉对象或改写译文。
