# body.anchored_blocks：裁决与修复规则

## 分类专用 Finding

| Finding | 严重度 | 直接证据 | owner |
|---|---|---|---|
| `ANCHORED_BLOCKS_TEXT_OWNERSHIP_INCOMPLETE` | HARD | 原生文字未恰好归属一个容器或保护集合 | `anchored_blocks_template_builder` |
| `ANCHORED_BLOCK_TEXT_OVERFLOW` | HARD | 译文无法装入容器允许区域 | `anchored_blocks_layout_planner` |
| `ANCHORED_BLOCK_CELL_OVERFLOW` | HARD | 卡片/单元文字越过本单元边界 | `anchored_blocks_layout_planner` |
| `ANCHORED_BLOCK_TEXT_COLLISION` | HARD | 译文与同页其他 owner 或保护对象相交 | `anchored_blocks_layout_planner` |
| `ANCHORED_BLOCK_STYLE_SCALE_DRIFT` | HARD | 同样式字号不一致或标题/正文字号层级漂移 | `anchored_blocks_layout_planner` |
| `ANCHORED_BLOCK_UNREADABLE_WRAP` | HARD | 断行产生不可读的窄列或碎片 | `anchored_blocks_layout_planner` |
| `ANCHORED_BLOCK_LOCKED_OBJECT_CHANGED` | HARD | source/candidate 锁定对象指纹不同 | `anchored_blocks_pdf_renderer` |
| `ANCHORED_BLOCK_PROTECTED_TEXT_CHANGED` | HARD | 保护文字内容或 bbox 改变 | `anchored_blocks_quality_judge` |
| `ANCHORED_BLOCK_OUTSIDE_ALLOWED_REGION_CHANGED` | HARD | 允许区域外存在渲染变化 | `anchored_blocks_pdf_renderer` |
| `ANCHORED_BLOCK_SOURCE_TEXT_REMAINED` | HARD | 已翻译容器仍残留原文 | `anchored_blocks_quality_judge` |
| `FONT_NOT_EMBEDDED` | HARD | 候选缺少目标字体资源 | `anchored_blocks_pdf_renderer` |

## Product PASS

- 候选 PDF 存在，过程链与产品裁决均通过；
- ownership 完整，块边界、锚点、顺序、对齐和字号层级保持；
- 译文全部位于本容器/卡片边界内，无溢出、碰撞、乱码或原文残留；
- 页码、数值及其他保护对象内容与位置不变；
- 字体已嵌入，允许区域外像素无变化。

## CAPABILITY FAILED

- 页面缺少原生文字或缺少可翻译语义文字；
- Provider/API key/字体不可用；
- Provider 丢失容器、译文不完整、产生占位符或删除必保字面量；
- 无法构造不侵入保护对象和相邻 owner 的安全回填区域。

## Repair 允许范围

只允许在目标容器 `allowed_bbox` 内调整换行、段距、行距、字号和该容器 placement；不得改译文语义、移动相邻块、扩大到全页，或改动图片、边框、页码及其他保护对象。

## Repair 接受

目标 Finding 改善、无新增硬问题、锁定对象保持、完整复裁通过时才接受；否则回滚。
