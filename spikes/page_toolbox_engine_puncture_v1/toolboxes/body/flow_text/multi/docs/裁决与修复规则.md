## Repair 允许范围

只允许修改当前 leaf 声明的 PagePatch 参数。

页眉/页脚自然语言不受源字形窄 bbox 的宽度上限约束。规划器优先在当前页同行锁定对象切分出的原属安全区段内扩宽；同一区段的多个可翻译片段必须保持顺序协同重排。若局部分区仍导致短标签或公司名断行，而相邻区段存在可容纳整组译文的大空白，可保持组内顺序迁入位移最小的可行区段；迁移组与锁定对象之间至少保留 `0.35em` 可见间隔。页码、年份、期号、logo 和独立图形不得移动或被覆盖；若整行仍无安全区段可避免拉丁词内断行，则保留 HARD Finding，不接受纵向碎裂回填。

## Repair 接受

目标 Finding 改善、无新增硬问题、锁定对象保持、完整复裁通过时才接受；否则回滚。
