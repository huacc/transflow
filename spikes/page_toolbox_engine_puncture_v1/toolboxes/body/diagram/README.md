# body.diagram toolbox

P14 的单叶工具箱。它只处理分类上游已经确定为 `body.diagram` 的单页 PDF。

运行时从 PDF 直接事实恢复可编辑原生文字、节点边界和连线端点。节点、连线、箭头、图片、颜色和空间关系均锁定；译文只允许写回明确 owner 的安全区域。正文标题、说明、页眉页脚可使用当前页证明安全的正文宽度和纵向留白；地图内部卡片、图例和锚点仍保持原图语义映射。图片内部文字不做 OCR，也不翻译。

验证顺序固定为一个代表页、三个结构扩展页、全部分类样本。第 36 轮最终统一复放为 30/30 `PAGE_PASSED`，且成功或失败路径都必须产出 `candidate.pdf`。当前样本在冻结前已做过缩略图盘点，因此结论为 `PASS_NON_BLIND`：可作为已知分类池回归证据，不能用于正式盲晋级。
