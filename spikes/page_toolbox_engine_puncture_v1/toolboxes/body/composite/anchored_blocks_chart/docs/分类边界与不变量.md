# 分类边界与不变量

## 正类

页面同时存在至少一个可独立识别的锚定块 owner，以及至少一个带原生可译语义文字的实际图表区域。锚定块可表现为 KPI 卡片、事项卡、业务模块或与图表并列的独立说明块；图表必须含柱、线、点、扇区、坐标轴、图例或数据图形之一。

## 非本类

- 只有图表和图例、没有独立锚定块：交给 `body.chart`；
- 只有独立卡片、没有实际图表语义文字：交给 `body.anchored_blocks`；
- 图表与结构化表格共同占主导：交给 `body.composite.chart_table`；
- 所有可译文字都被 P13 图表语义 owner 占有：P15 显式 `P11_TRANSLATABLE_OWNER_NOT_FOUND`，禁止伪造锚定块。

## 不变量

1. 每个原生文本对象恰好一个 owner：`anchored`、`chart`、`shared` 或 `protected`。
2. 图表图片、矢量图形、轴线、点、柱、扇区、数值、数据标签、颜色和几何保持锁定。
3. P11 块边界与 P13 图表安全区互不借用；跨 owner 碰撞只能失败，不能挪用对方空间。
4. 全页只发一个翻译请求，统一 ID 带 owner 前缀，回切时恢复叶工具原 ID。
5. 只做一次 PDF redaction/write，禁止先写 P11 再写 P13。
6. 固定译文只构成机械证据，不构成产品翻译证据。
