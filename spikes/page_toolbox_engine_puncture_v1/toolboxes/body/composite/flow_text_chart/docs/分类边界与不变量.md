# 分类边界与不变量

## 处理边界

本工具箱只接受 `body.composite.flow_text_chart`。页面必须同时存在可由直接 PDF 结构事实识别的实质正文流和主体图表原生文字。纯正文、纯图表、表格、示意图、锚定卡片及 `body.freeform` 不在本工具箱能力内。

图片化图表内部文字不进入翻译请求；不使用 OCR 回填。图表中的图片、矢量构件、轴线、色块、柱线点扇区、数值刻度和几何关系始终锁定。

## 所有权

每个原生文字对象恰好属于一个集合：

- `flow`：构成实质叙述正文或正文小标题，只能在所属 FlowBand 回流；
- `chart`：与图表视觉区域有结构关联的标题、图例、类别标签、轴标签或注释；
- `shared`：页级标题、页眉页脚、来源等不应被正文吸收的共享文字；
- `protected`：数值刻度、图片内文字对应物、装饰或没有安全翻译 owner 的原生文字。

所有权必须穷尽、互斥，并以对象 ID 和稳定容器 ID 落盘。距离只能作为结构证据之一，不能单独决定图注、来源或短说明的 owner。

## 不变量

- 正文的阅读顺序、语义段落和所属栏道保持；行首不得出现闭合标点；
- 同一父级或相邻源块的语义续接保持在同一正文栏道；图表 owner 移除后要再合并满足垂直邻接、字号兼容和有限左缘抖动的普通字重续接，粗体标题不得借此并入正文；页脚或脚注只有与正文栏道横向相交时才限制 FlowBand；
- 正文输出框不得越出 FlowBand，不得进入图表 guard region；
- 图表标签不得进入正文流，标签与图例/系列的关联保持；
- 一个页面只产生一个有序翻译请求，不重复、不遗漏当前目标语言所需的原生语义容器；
- 不同源文容器不得共享同一份长译文；命中跨容器长译文重复时必须由翻译提供方失败，不能进入布局；
- 所有通过页只从源 PDF 做一次渲染，页数、页面尺寸和不可动对象保持；
- 最小字号仍无法安全容纳时返回能力不足，不移动图表、不缩到不可读、不把源页复制品当候选；
- 当前 32 页都没有出现同一 FlowBand 内的并行多栏正文直接证据，因此本轮实际选择均为 `single`；这不等于宣称 P5 已通过或 P17 已具备形式晋级条件。
