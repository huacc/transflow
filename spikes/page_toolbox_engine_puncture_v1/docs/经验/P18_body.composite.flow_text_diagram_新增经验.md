# P18 `body.composite.flow_text_diagram` 新增经验

本文只记录 P18 相对既有 P4、P13、P14、P16 经验的净新增结论。页眉页脚语义翻译、目标语言自然换行、源字形 bbox 不是硬边界、纵向留白的有限字号/行距回升、列表标记属于结构、实际字形验收和截断后的语义分段重试，均沿用既有经验，不在这里重复。

## 1. 排版方向是框架组级合同，不是逐文字框的独立选择

同一组窄长节点、同一层级泳道或同一种组织框架中的源文字如何排布，译文就应继承同一种方向。源页采用竖排时，组内译文全部使用竖排；单列放不下可以变成两列竖排，但不能让某些框竖排、另一些框横排。源页采用横排时，也不能只因为译文较长就把单个框擅自旋转。

方向判断应来自源文字的行栈、字形框长宽比、节点长宽比和同组关系。方向确定后，再在该方向内搜索列数、字号、行距和局部安全框。字符数、语言方向、样本名、固定文句或绝对坐标都不能直接决定旋转。

> 先继承源框架方向，再解决目标语言 fit；不能用逐框 fit 反过来改写框架语法。

## 2. 复合页必须先恢复语义原子，再做一次页级翻译

PDF 抽取块既可能把一个自然段拆成多条视觉行，也可能把多个独立节点标签压成一个块。P18 不能把抽取块直接当翻译单元，而应先按 owner 和局部结构恢复语义原子：

- 同一正文段落或同一节点说明的连续碎片合成一个单元，源视觉行尾不进入译文硬约束；
- 同一节点中上下堆叠但各自有独立框的标签继续保持多个局部 frame，不能合并成一段；
- 同一源行若 span 序号中断且两侧存在可见间隔，说明中间可能有被保护的分隔符；分隔符两侧必须保留独立 frame，并分别在自己的安全区段内对齐；
- 列表标题、每个项目符号及其后续多行文字分别形成稳定单元，标记与对应语义绑定；
- 正文、节点文字、共享标题/图注和 margin 语言合并为一次页级请求，再按稳定 container ID 回填。

这样才能同时避免逐视觉行强制换行、多个标签共享一份整段译文、列表语义串行和跨 owner 重复翻译。

## 3. 节点内重排要同时尊重内容起点和结构阻挡物

节点安全内框只是最大可用区域。列表或多标签组重新排布时，不能一律从安全框顶部开始，否则会压住节点标题、色带或中央说明。应保留源内容组的起始带，并把标题栏、独立说明和相邻 owner 的实际字形框视为 blocker；只有源内容起点已进入 blocker 时，才把整组移动到 blocker 下方。

组内排布使用统一方向和字号层级，允许整体下移、自然换行或在可证明的空白中扩展。单个成员不得脱离组独自抢占相邻标签空间。

## 4. 旋转文字的产品包络必须由旋转后的真实字形计算

竖排实现通常先在横向临时框中试排，再旋转写入 PDF。此时临时 textbox、节点槽位和最终可见字形是三个不同几何对象。若碰撞门只检查未旋转的槽位，会误报相邻竖排列相撞；若只检查最大允许框，又会漏掉旋转后字形越过分隔线。

稳定做法是：以目标字体和最终字号完成试排，按实际旋转角度变换字形包围盒，再用最终 glyph bbox 检查页面边界、同组列间距、标题/图注和连接线。两列竖排也必须逐列生成互不相交的最终字形包络。

## 5. 私有字体符号要通过真实 PDF 写入和提取探针归一

Wingdings 等字体中的项目符号或箭头可能以私有区码点出现。字体 cmap 声称覆盖某个 Unicode 箭头，并不保证当前 PDF 引擎能稳定写入、显示和提取；只做字体覆盖检查仍可能得到空白、方框或 `NUL`。

归一符号前应使用与生产相同的字体、PDF 写入 API 和提取链路做最小探针。替代字符必须同时满足可见、可提取、语义类别一致，例如箭头仍映射为可用箭头，而不是无条件降级为圆点。归一后的标记继续进入 required literal 和结构门禁。

## 6. 翻译校验失败必须贯通到页级终态

翻译守卫即使正确发现句尾截断、括号不闭合、列表标记丢失或分句边界缺失，如果 engine 汇总时漏接某类 Finding，页面仍可能错误进入 `PAGE_PASSED`。新增校验规则时必须同时补三层回归：守卫能够报错、页引擎把 Finding 归入能力失败、批次汇总不再把该页计入产品通过。

结构化输出导致长单元反复截断时，可先去掉仅承担装饰作用的成对引号后完整重试；仍失败才按语义标点拆分，并逐分句请求、逐分句校验后合并。括号内容、列表标记、日期、数字和其他结构字面量不能随清洗丢失。

## 7. 机械通过后仍要检查真实候选，视觉失败要反哺门禁

本轮多次出现字符串、布局框和批次状态均通过，但真实 PDF 仍可看到截断、私有符号方框或错误方向。关闭问题前至少要检查高风险页的真实 `candidate.pdf` 或原分辨率预览，而不是只看 JSON；发现可见失败后，应先补可复现测试和机械门禁，再重跑全量。

PDF textbox 的默认 CJK 换行还可能把逗号、句号、右括号等闭合标点放到行首。不能插入字体不支持的零宽字符掩盖问题；应以生产字体和最终 bbox 实际排版取行，优先尝试不改变行距和结构层级的最小字号微调。仍无法消除禁则换行时必须生成 hard Finding，不能机械通过。

窄节点的 textbox 整体能容纳译文，不等于单个拉丁词能完整容纳。应使用目标字体实测最长词的宽度，先在可读下限内统一缩小字号；到达下限仍放不下时必须硬失败，不得产生 `Compensatio / n`这类半词换行。

工程产品证据与形式晋级继续分开。即使当前已知池全部通过，前置工具箱 Gate、workflow freeze 和严格盲测不满足时，P18 仍只能保持 `EXPERIMENTAL`，不得生成 `promotion_manifest.json`。

## 8. 放宽源 bbox 必须同时给出相邻槽位不碰撞证明

源 bbox 不是译文的硬边界，但“可向空白扩展”也不是无条件放大。短横排节点、页眉页脚和分隔符两侧的标题扩展后，必须用最终字形框重新检查同组平行槽位、相邻 owner、受保护分隔符和页面安全边界。源槽位原本不相交而候选扩展后会相交时，应放弃该次扩展，改用组内统一字号、自然换行或原方向分列；不能让单个短标签抢占相邻标签的区域。

受保护分隔符一侧存在页面边缘空白时，可以只把该侧文字框延伸到页面安全边界，不移动分隔符，也不合并两侧语义。兄弟标题应共享同一字号层级，避免一侧因为空间较大而显著放大、另一侧被迫缩小。

项目符号和箭头要与其后的首个语义词绑定为一个不可拆起始单元；换行后的续行对齐正文起点，而不是把标记孤立到上一行或页面边缘。标记位置可以随整段安全重排，但不能脱离对应语义。

## 9. 原生图片是正文重排的硬阻挡物

复合页中的证书、照片、徽标和作为背景以外用途的位图，不能因为不属于文本 owner 就被正文覆盖。正文在图片上方扩展时，应先保留字号、行距和语义顺序，利用图片前已有的纵向空白对尾部行做最小向上紧凑；只有全部最终字形仍停在图片安全边界之外时才接受布局。任何新增 flow/shared 字形与原生图片相交都应产生 hard Finding，而不是依赖人工抽检发现。

## 10. CID 字体恢复不能改写页面中可信的重复缩写

Identity-H、Adobe-CNS1 等 CID 字体的字符恢复可能把原本可见的 ASCII 缩写错误映射成另一串字母。若同一全大写缩写在页面普通字体事实中重复出现，应把它视为可信页面字面量，在 CID 恢复时按完整 ASCII run 保留；与中文相邻的短大写缩写还应进入 required literal 校验。不能对 ASCII run 逐字符盲目套用 CID 映射，否则会出现视觉源文正确、模板事实和译文却被悄悄改写的能力缺口。

## 11. PDF 图片对象要区分内容图片与源文字背景

不能把所有 image object 一律当成正文硬阻挡物。页顶或页底可能存在横贯页面的装饰底图，源文字本来就绘制在该图上；若仍把它当内容图片，候选会产生大量假碰撞并被错误压缩。稳定判据是同时检查图片是否贴近页边、是否近乎横贯整页，以及源文字字形是否已经与图片相交。满足这些条件的对象可视为文字背景；证书、照片、徽章等源文字原本不覆盖的图片继续作为硬阻挡物。

该分类只豁免背景图自身的碰撞，不能放宽页面边界或相邻文本 owner。任何原本位于内容图片侧边的段落，在横向扩展时只能使用图片左侧或右侧的连续安全通道。

## 12. 弹性结构带必须在最终几何变化后再统一重排

标题展宽、前缀回接、字号回升和图片侧通道裁剪都会改变最终文本高度。若只在这些变化之前做一次结构带排布，后续单个容器可能重新侵入上一段正文、分隔线或图片。复合页应在所有横向几何变化完成后，对同一结构区执行最后一次组级重排，并保留已经得到的字号上限，避免第二遍又把文字放大。

侧栏结构带的纵向安全区不能只取 owner 的最大 allowed bbox。上界至少是该语义组在源页的首行起点，下界是与该侧栏横向相交的最近内容图片上沿；整组只在这个闭区间内按原语义顺序压缩或展开。空间不足时应继续搜索组级字号和行距，仍无法容纳才硬失败，不能把尾部逆序向上顶进页眉或上一段正文。

## 13. 页眉页脚拉平只改几何，不改语义属性

将页眉页脚压成单行时，空白归一化很容易顺带吞掉章节编号后的受保护间隔，覆盖原有左右对齐策略，或把分隔符两侧标题重新解释成一个框。拉平步骤必须保留 `preserved_prefix`、左右对齐、语义 frame 和必需字面量，只允许调整连续安全区段、字号、单行行高和 bbox。几何优化后还要重新跑前缀、分隔符、相邻槽位和页码碰撞回归。
