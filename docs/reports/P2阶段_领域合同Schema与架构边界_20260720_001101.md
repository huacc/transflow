## 9. 设计问题与处理

- JSON Schema 无法独立表达“返回 unit_id 与请求集合及顺序完全一致”的跨数组关系；解决方案是在 JSON Schema 外由 `TranslationBundle` 领域构造器强制校验，并以 P2.1-T03 覆盖全部漂移类型。
- spike 的 PASS 不等于生产晋升。Catalog 启用条件固定为 `evidence_status=PASS` 且 `promotion_manifest_present=true`；当前 17 条路由均未满足，因此全部禁用。
- P2 没有调用任何真实模型，也没有使用或保存模型密钥；这符合本阶段只冻结两个 AI Port 合同、不实现外部 AI Adapter 的边界。

最终结论：P2 全部二级子计划和 Gate G2 技术验收通过，可以进入 P3。
